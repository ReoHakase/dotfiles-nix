# Git

`~/.config/git/config` is generated by Home Manager from `programs.git` in `home/common.nix` (credential helpers use `${pkgs.gh}` / `${pkgs.gnupg}`, not store paths in git).

Host-specific signing keys are not tracked. Put them in `~/.config/git/local`, which is included by the generated config:

```ini
[user]
	signingKey = <host-specific-key-id>
```
